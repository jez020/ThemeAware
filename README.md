
# ThemeAware

An Electron desktop app built with React and TypeScript.

## Features

- **Electron + React + TypeScript** - Cross-platform desktop architecture with a modern UI stack
- **Theme-Aware Desktop Experience** - Native desktop behavior with renderer and main process separation
- **Fast Development Workflow** - Hot reload and webpack-based dev tooling

## Installation

```bash
npm install
```

## Quick Start

```bash
# Start the app in development mode
npm start

# Create a production package
npm run package
```

## Requirements

- Node.js 18+
- npm 9+
- macOS, Windows, or Linux
